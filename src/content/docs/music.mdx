## 각 항목 태그
| 태그 | 제목 | 설명 | 비고 |
| :--- | :------- | :------------ | :--- |
| t | 음악/영상 타이틀 | 음악 또는 영상의 제목을 적어주세요 |
| u | 아티스트/업로더 | 해당 콘텐츠를 올린이의 닉네임/이름을 적어주세요 |
| remixed | 리믹스됐나요? | "`0`: 원곡임, `1`: 리믹스됨" 둘 중 한 값을 표기하세요 | 비고에는 원곡에서 가져와 사용된 소재를 적으세요. 없을 경우 `none` 표기합니다. |
| sampling | 샘플링 | "`0`: 원곡임, `1`: 샘플링됨" 샘플링을 한 음악인가요? | 비고에는 사용된 샘플링 소재를 적으세요. 없을 경우 `none` 표기합니다. |
| g | 장르 | 해당 음악의 알맞거나 근접하는 장르를 적어주세요 |
| m | 분위기 | 해당 음악에 맞는 분위기를 찾아 적어주세요 |
